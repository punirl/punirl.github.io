LEAF (Lightweight Embedded Audio Framework)

An audio library designed to be lightweight and give control over memory usage and performance for embedded platforms that have limitations in those areas. Written in straight C for portability, with no dependencies. Can be included in JUCE projects for computer and mobile device development as well. (Formerly known as OOPS).
